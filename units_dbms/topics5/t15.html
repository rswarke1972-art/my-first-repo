Indexing and Aggregation

Indexing improves query performance in MongoDB, while Aggregation is used to process and analyze data.

Indexing

  • Speeds up data retrieval
  • Created on specific fields

Example

db.student.createIndex({name: 1})

Aggregation

  • Processes data records
  • Used for grouping and calculations

Example

db.student.aggregate([ {$group: {_id: "$age", count: {$sum: 1}}} ])

Importance

  • Improves performance
  • Helps in data analysis

Conclusion

Indexing enhances performance, while aggregation supports data processing in MongoDB.
