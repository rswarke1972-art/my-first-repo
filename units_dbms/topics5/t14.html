MongoDB: CRUD Operations

CRUD stands for Create, Read, Update, and Delete. These are the basic operations performed in MongoDB.

1. Create

  • Insert new document into collection

Example

db.student.insertOne({name: "Manoj", age: 20})

2. Read

  • Retrieve documents from collection

Example

db.student.find()

3. Update

  • Modify existing document

Example

db.student.updateOne({name: "Manoj"}, {$set: {age: 21}})

4. Delete

  • Remove document from collection

Example

db.student.deleteOne({name: "Manoj"})

Conclusion

CRUD operations are fundamental for managing data in MongoDB.
