ER Diagram

An ER (Entity Relationship) Diagram is a graphical representation of entities, attributes, and relationships in a database system.

Components of ER Diagram

1. Entity

  • Represented by a rectangle
  • Example: Student, Employee

2. Attribute

  • Represented by an ellipse
  • Connected to entity

3. Relationship

  • Represented by a diamond shape
  • Connects two or more entities

4. Primary Key

  • Underlined attribute
  • Uniquely identifies an entity

Types of Relationships

  • One-to-One (1:1)
  • One-to-Many (1:N)
  • Many-to-Many (M:N)

Conclusion

ER Diagrams help visualize database structure and simplify database design.
