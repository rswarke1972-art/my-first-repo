View of Data

In a database system, data can be viewed at different levels of abstraction. These levels hide complex details and make the system easier to use.

Levels of Data Abstraction

1. Physical Level

  • Lowest level of abstraction
  • Describes how data is stored in memory
  • Includes storage structures and indexing

2. Logical Level

  • Describes what data is stored
  • Describes relationships among data
  • Used by database administrators

3. View Level

  • Highest level of abstraction
  • Describes only part of the database
  • Used by end users

Data Independence

  • Physical Data Independence
  • Logical Data Independence

Conclusion

View of Data allows users to access information at different levels while hiding unnecessary implementation details.
