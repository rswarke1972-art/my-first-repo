Database System Structure

The Database System Structure describes the overall organization of a DBMS and its components. It shows how users interact with the database.

Main Components

1. Database

  • Collection of related data
  • Stored in organized form

2. DBMS Software

  • Manages data storage and retrieval
  • Provides security and integrity

3. Application Programs

  • Used by end users
  • Interact with DBMS

4. Users

  • Database Administrator (DBA)
  • Application Programmers
  • End Users

Functional Components of DBMS

  • Query Processor
  • Storage Manager
  • Transaction Manager

Conclusion

Database System Structure defines how data, software, and users interact within a DBMS.
