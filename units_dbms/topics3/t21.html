Boyce-Codd Normal Form (BCNF)

BCNF (Boyce-Codd Normal Form) is a higher level of normalization than 3NF. It removes certain anomalies that 3NF does not handle.

Definition

  • A relation is in BCNF if for every functional dependency A → B, A is a super key.

Difference Between 3NF and BCNF

  • 3NF allows some exceptions
  • BCNF is stricter than 3NF

Example

Course (Teacher, Subject)

  • If Teacher → Subject and Subject → Teacher
  • Both are candidate keys

Importance

  • Eliminates redundancy
  • Prevents anomalies

Conclusion

BCNF ensures that every determinant in a functional dependency is a super key.
