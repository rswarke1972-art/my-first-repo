Basic Concept - Relational Model

The Relational Model is a database model that organizes data into tables (relations). It was proposed by E.F. Codd in 1970.

Main Concepts

1. Relation

  • A table in a database
  • Consists of rows and columns

2. Tuple

  • A single row in a table

3. Attribute

  • A column in a table

4. Domain

  • Set of allowed values for an attribute

Example

Student (Roll_No, Name, Age)

Advantages

  • Simple structure
  • Easy to understand
  • Data independence

Conclusion

The relational model organizes data in tabular form, making database management simple and efficient.
