Decomposition using Functional Dependencies

Decomposition is the process of dividing a relation into two or more smaller relations to remove redundancy and anomalies. It is based on Functional Dependencies (FD).

Functional Dependency (FD)

  • If A → B, then A determines B
  • A is called determinant
  • B is dependent attribute

Need for Decomposition

  • Remove redundancy
  • Eliminate update anomalies
  • Improve data integrity

Types of Decomposition

  • Lossless Decomposition
  • Lossy Decomposition

Lossless Decomposition

  • No information is lost
  • Original table can be reconstructed using JOIN

Conclusion

Decomposition using functional dependencies helps in achieving higher normal forms.
