Join Queries

Join queries are used to retrieve data from two or more tables based on a related column between them.

Types of Joins

1. INNER JOIN

  • Returns matching records from both tables

Example

SELECT Student.Name, Course.Course_Name FROM Student INNER JOIN Course ON Student.Course_ID = Course.Course_ID;

2. LEFT JOIN

  • Returns all records from left table and matching from right table

3. RIGHT JOIN

  • Returns all records from right table and matching from left table

4. FULL JOIN

  • Returns all records when there is a match in either table

Conclusion

Join queries help combine data from multiple tables based on relationships between them.
