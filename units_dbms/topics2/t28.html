Stored Procedure and Function

Stored Procedures and Functions are PL/SQL blocks that are stored in the database and can be executed whenever required.

Stored Procedure

  • Performs a specific task
  • May or may not return a value

Example

CREATE OR REPLACE PROCEDURE show_msg IS BEGIN DBMS_OUTPUT.PUT_LINE('Hello Manoj'); END;

Function

  • Always returns a value

Example

CREATE OR REPLACE FUNCTION get_age RETURN NUMBER IS BEGIN RETURN 20; END;

Advantages

  • Code reusability
  • Improved security
  • Better performance

Conclusion

Stored Procedures and Functions help organize reusable program logic in PL/SQL.
