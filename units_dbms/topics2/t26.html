Control Statement (PL/SQL)

Control statements in PL/SQL are used to control the flow of execution in a program. They allow decision making and looping.

Types of Control Statements

1. IF Statement

  • Used for decision making

Example

IF age > 18 THEN DBMS_OUTPUT.PUT_LINE('Eligible'); END IF;

2. IF-ELSE Statement

  • Executes alternative block if condition is false

3. LOOP Statement

  • Repeats statements multiple times

Example

LOOP EXIT WHEN counter > 5; counter := counter + 1; END LOOP;

4. FOR LOOP

  • Repeats fixed number of times

5. WHILE LOOP

  • Repeats while condition is true

Conclusion

Control statements help manage program logic and execution flow in PL/SQL.
