Indexes

An Index in SQL is used to improve the speed of data retrieval operations on a table. It works like an index in a book, allowing faster access to rows.

Purpose of Index

  • Improve query performance
  • Reduce search time

Creating an Index

CREATE INDEX index_name ON table_name (column_name);

Example

  • CREATE INDEX idx_name ON Student(Name);

Types of Indexes

  • Primary Index
  • Unique Index
  • Composite Index

Dropping an Index

  • DROP INDEX index_name;

Conclusion

Indexes enhance database performance by speeding up data retrieval.
