DCL (Data Control Language)

DCL (Data Control Language) is used to control access to data in a database. It manages permissions and security of database objects.

Main DCL Commands

1. GRANT

  • Gives privileges to users
  • Example: GRANT SELECT ON Student TO User1;

2. REVOKE

  • Removes privileges from users
  • Example: REVOKE SELECT ON Student FROM User1;

Privileges

  • SELECT
  • INSERT
  • UPDATE
  • DELETE

Conclusion

DCL commands help maintain database security by controlling user access.
