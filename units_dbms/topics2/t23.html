Nested Queries

A Nested Query (Subquery) is a query inside another SQL query. It is used to retrieve data that will be used by the main query.

Types of Nested Queries

1. Single Row Subquery

  • Returns only one row

Example

SELECT * FROM Student WHERE Age = (SELECT MAX(Age) FROM Student);

2. Multiple Row Subquery

  • Returns multiple rows
  • Used with IN, ANY, ALL operators

Example

SELECT * FROM Student WHERE Course_ID IN (SELECT Course_ID FROM Course WHERE Credits > 3);

Importance

  • Improves query flexibility
  • Used for complex filtering

Conclusion

Nested queries allow one query to use the result of another query.
