DDL (Data Definition Language)

DDL (Data Definition Language) is used to define and modify the structure of database objects such as tables, views, and indexes.

Main DDL Commands

1. CREATE

  • Used to create database objects
  • Example: CREATE TABLE Student (...);

2. ALTER

  • Used to modify existing table structure
  • Example: ALTER TABLE Student ADD Age NUMBER;

3. DROP

  • Used to delete database objects
  • Example: DROP TABLE Student;

4. TRUNCATE

  • Removes all records from table
  • Table structure remains

5. RENAME

  • Used to rename a table

Features

  • Automatically commits changes
  • Affects database structure

Conclusion

DDL commands are used to define and manage database structure efficiently.
