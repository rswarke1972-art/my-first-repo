Date and Numerical Functions

Date and Numerical functions in SQL are used to perform operations on date values and numeric data.

Date Functions

1. SYSDATE

  • Returns current system date
  • Example: SELECT SYSDATE FROM DUAL;

2. ADD_MONTHS()

  • Adds months to a date

3. MONTHS_BETWEEN()

  • Finds difference between two dates

4. NEXT_DAY()

  • Returns next specified weekday

Numerical Functions

1. ROUND()

  • Rounds a number

2. TRUNC()

  • Removes decimal part

3. MOD()

  • Returns remainder of division

4. ABS()

  • Returns absolute value

Conclusion

Date and numerical functions help perform calculations and date operations in SQL.
