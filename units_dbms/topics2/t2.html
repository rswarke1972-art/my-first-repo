Set

In SQL, a set represents a collection of rows returned by a query. Set operations are used to combine results from multiple SELECT statements.

Properties of Sets in SQL

  • Each SELECT query returns a set of rows
  • Columns must have same number and data types for set operations
  • Duplicate rows may be removed depending on operation

Example

SELECT Name FROM Student;

Conclusion

A set in SQL refers to the result of a query, which can be combined with other sets using set operations.
