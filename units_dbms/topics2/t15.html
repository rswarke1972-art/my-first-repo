Aggregate Functions

Aggregate functions in SQL are used to perform calculations on a group of values and return a single result.

Common Aggregate Functions

1. COUNT()

  • Counts number of rows
  • Example: SELECT COUNT(*) FROM Student;

2. SUM()

  • Returns total of numeric column

3. AVG()

  • Returns average value

4. MAX()

  • Returns highest value

5. MIN()

  • Returns lowest value

Important Points

  • Ignore NULL values
  • Often used with GROUP BY clause

Conclusion

Aggregate functions help summarize data efficiently in SQL queries.
