Cursor (PL/SQL)

A Cursor in PL/SQL is used to retrieve and process multiple rows returned by a SELECT statement. It allows row-by-row processing of query results.

Types of Cursors

1. Implicit Cursor

  • Automatically created by Oracle
  • Used for single-row queries

2. Explicit Cursor

  • Declared by programmer
  • Used for multi-row queries

Steps in Using Explicit Cursor

  • Declare cursor
  • Open cursor
  • Fetch rows
  • Close cursor

Example

DECLARE CURSOR c1 IS SELECT Name FROM Student; BEGIN OPEN c1; FETCH c1 INTO v_name; CLOSE c1; END;

Conclusion

Cursors allow efficient row-by-row processing of query results in PL/SQL.
