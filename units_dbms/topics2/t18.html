Group By and Having Clause

The GROUP BY clause is used to group rows that have the same values in specified columns. The HAVING clause is used to filter groups based on conditions.

GROUP BY

  • Used with aggregate functions
  • Groups rows with same values

Example

SELECT Age, COUNT(*) FROM Student GROUP BY Age;

HAVING Clause

  • Used to filter grouped data
  • Works with GROUP BY

Example with HAVING

SELECT Age, COUNT(*) FROM Student GROUP BY Age HAVING COUNT(*) > 1;

Difference Between WHERE and HAVING

  • WHERE filters rows before grouping
  • HAVING filters groups after grouping

Conclusion

GROUP BY and HAVING are used together to organize and filter grouped data.
