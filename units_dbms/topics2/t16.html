View

A View in SQL is a virtual table based on the result of a SELECT query. It does not store data physically, but displays data from one or more tables.

Creating a View

CREATE VIEW view_name AS SELECT column_name FROM table_name;

Example

  • CREATE VIEW Student_View AS SELECT Name, Age FROM Student;

Advantages

  • Improves security
  • Simplifies complex queries
  • Hides sensitive data

Dropping a View

  • DROP VIEW view_name;

Conclusion

Views provide a simplified and secure way to access data from database tables.
