DML (Data Manipulation Language)

DML (Data Manipulation Language) is used to manage data stored inside database tables. It allows inserting, updating, deleting, and retrieving records.

Main DML Commands

1. INSERT

  • Adds new records into a table
  • Example: INSERT INTO Student VALUES (1, 'Manoj', 20);

2. UPDATE

  • Modifies existing records
  • Example: UPDATE Student SET Age = 21 WHERE Roll_No = 1;

3. DELETE

  • Removes records from a table
  • Example: DELETE FROM Student WHERE Roll_No = 1;

4. SELECT

  • Retrieves data from table

Features

  • Does not change table structure
  • Can be controlled using TCL commands

Conclusion

DML commands are essential for handling data inside database tables.
