String Functions

String functions in SQL are used to manipulate character data stored in database tables.

Common String Functions

1. UPPER()

  • Converts text to uppercase
  • Example: SELECT UPPER(Name) FROM Student;

2. LOWER()

  • Converts text to lowercase

3. LENGTH()

  • Returns length of string

4. SUBSTR()

  • Extracts part of a string

5. CONCAT()

  • Joins two strings

6. TRIM()

  • Removes extra spaces

Conclusion

String functions help manipulate text data efficiently in SQL queries.
