Set Membership

Set membership in SQL is used to test whether a value belongs to a specified set of values. It is commonly used with the IN and NOT IN operators.

IN Operator

  • Checks if a value matches any value in a list

Example

SELECT * FROM Student WHERE Age IN (18, 19, 20);

NOT IN Operator

  • Checks if a value does not match any value in a list

Example

SELECT * FROM Student WHERE Age NOT IN (18, 19);

Importance

  • Simplifies multiple OR conditions
  • Used in nested queries

Conclusion

Set membership operators help filter data based on a list of values efficiently.
