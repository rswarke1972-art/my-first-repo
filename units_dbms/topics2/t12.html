Select Queries

SELECT statement is used to retrieve data from one or more tables in a database. It is the most commonly used SQL command.

Basic Syntax

SELECT column_name FROM table_name;

Examples

  • SELECT * FROM Student;
  • SELECT Name, Age FROM Student;

Using WHERE Clause

  • SELECT * FROM Student WHERE Age > 18;

Using ORDER BY

  • SELECT * FROM Student ORDER BY Name;

Using DISTINCT

  • SELECT DISTINCT Age FROM Student;

Conclusion

Select queries are used to retrieve and filter data from database tables efficiently.
