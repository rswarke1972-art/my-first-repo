Basic HTML

HTML (Hyper Text Markup Language) is the standard language used to create web pages.

Basic Structure

  • <!DOCTYPE html>
  • <html>
  • <head>
  • <body>

Common Tags

  • <h1> to <h6> (Headings)
  • <p> (Paragraph)
  • <a> (Link)
  • <img> (Image)

Conclusion

Basic HTML forms the foundation of all web pages.
