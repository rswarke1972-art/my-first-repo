Text Formatting on Web Pages

HTML provides various tags to format text and improve the appearance of web content.

Common Formatting Tags

  • <b> - Bold text
  • <i> - Italic text
  • <u> - Underline text
  • <strong> - Important text
  • <em> - Emphasized text
  • <br> - Line break

Purpose

  • Improve readability
  • Highlight important information

Conclusion

Text formatting enhances the presentation of web pages.
