Inserting Images

Images can be added to a web page using the <img> tag in HTML.

Basic Syntax

  • <img src="image.jpg" alt="Description">

Important Attributes

  • src - Specifies image path
  • alt - Alternative text
  • width - Image width
  • height - Image height

Conclusion

Images make web pages more attractive and informative.
