Loops

Loops in PHP are used to execute a block of code multiple times.

Types of Loops

  • for loop
  • while loop
  • do...while loop
  • foreach loop

Example

  • <?php
  • for ($i = 1; $i <= 5; $i++) {
  • echo $i;
  • }
  • ?>

Conclusion

Loops reduce repetition and improve program efficiency.
