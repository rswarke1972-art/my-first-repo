Decision Making Statements

Decision making statements in PHP are used to execute code based on specific conditions.

Common Statements

  • if statement
  • if...else statement
  • elseif statement
  • switch statement

Example

  • <?php
  • $age = 20;
  • if ($age > 18) {
  • echo "Adult";
  • }
  • ?>

Conclusion

Decision statements control program flow in PHP.
