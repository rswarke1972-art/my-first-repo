Arrays

Arrays in PHP are used to store multiple values in a single variable.

Types of Arrays

  • Indexed Array
  • Associative Array
  • Multidimensional Array

Example

  • <?php
  • $numbers = array(1, 2, 3);
  • echo $numbers[0];
  • ?>

Conclusion

Arrays help manage collections of related data in PHP.
