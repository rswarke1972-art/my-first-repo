Statements

Statements in PHP are instructions that tell the program what action to perform.

Types of Statements

  • Assignment statements
  • Conditional statements
  • Loop statements
  • Return statements

Example

  • <?php
  • $x = 10; // Assignment
  • echo $x; // Output statement
  • ?>

Conclusion

Statements form the basic building blocks of PHP programs.
