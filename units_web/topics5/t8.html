PHP Functions

Functions in PHP are blocks of code that perform a specific task and can be reused multiple times.

Advantages

  • Code reusability
  • Better organization
  • Easy maintenance

Example

  • <?php
  • function greet() {
  • echo "Hello World";
  • }
  • greet();
  • ?>

Conclusion

Functions improve program structure and readability.
