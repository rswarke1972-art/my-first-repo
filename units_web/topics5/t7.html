Operators

Operators in PHP are symbols used to perform operations on variables and values.

Types of Operators

  • Arithmetic Operators (+, -, *, /)
  • Comparison Operators (==, !=, >, <)
  • Logical Operators (&&, ||)
  • Assignment Operators (=, +=, -=)

Example

  • <?php
  • $a = 5;
  • $b = 3;
  • echo $a + $b;
  • ?>

Conclusion

Operators help perform calculations and logical operations in PHP.
