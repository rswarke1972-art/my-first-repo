External CSS

External CSS is written in a separate .css file and linked to HTML pages.

How It Works

  • Create a file named style.css
  • Write CSS rules in that file
  • Link it using <link> tag in HTML

Example

  • <link rel="stylesheet" href="style.css">

Advantages

  • Reusable across multiple pages
  • Easy maintenance

Conclusion

External CSS ensures consistent styling across an entire website.
