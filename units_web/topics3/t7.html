ID Selector

The ID selector in CSS is used to apply styles to a specific HTML element with a unique id.

Syntax

  • #idname { property: value; }

Example

  • <p id="para1">Hello</p>
  • #para1 { color: red; }

Features

  • Unique per page
  • High priority in CSS

Conclusion

ID selectors help style specific elements uniquely.
