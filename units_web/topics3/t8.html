Types of Selectors

CSS selectors are used to select HTML elements for applying styles.

Common Types

  • Element Selector (p, h1)
  • ID Selector (#idname)
  • Class Selector (.classname)
  • Universal Selector (*)
  • Group Selector (h1, p)

Purpose

  • Apply styles efficiently
  • Control page design

Conclusion

Selectors are essential for targeting elements in CSS.
