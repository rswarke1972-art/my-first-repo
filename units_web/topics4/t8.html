Prototypes

Prototypes in JavaScript allow objects to inherit properties and methods from other objects.

Purpose

  • Enable inheritance
  • Share methods between objects

Example

  • function Person(name) { this.name = name; }
  • Person.prototype.greet = function() {
  • console.log("Hello " + this.name);
  • };

Conclusion

Prototypes support object-oriented programming in JavaScript.
