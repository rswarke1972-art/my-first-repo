Flynn's Taxonomy

Flynn's Taxonomy is a classification system for computer architectures based on the number of instruction streams and data streams processed simultaneously.

Types of Flynn's Taxonomy

1. SISD (Single Instruction Single Data)

  • One instruction stream
  • One data stream
  • Example: Traditional single processor system

2. SIMD (Single Instruction Multiple Data)

  • One instruction stream
  • Multiple data streams
  • Used in vector processors

3. MISD (Multiple Instruction Single Data)

  • Multiple instruction streams
  • Single data stream
  • Rarely used architecture

4. MIMD (Multiple Instruction Multiple Data)

  • Multiple instruction streams
  • Multiple data streams
  • Used in multiprocessor systems

Conclusion

Flynn's Taxonomy helps classify parallel computer architectures based on how they process instructions and data.
