UMA Architecture

UMA (Uniform Memory Access) Architecture is a type of shared memory system where all processors have equal access time to the main memory.

Features

  • Shared memory system
  • Equal memory access time for all processors
  • Common bus connects processors and memory

Advantages

  • Simple design
  • Easy programming model
  • Efficient for small multiprocessor systems

Disadvantages

  • Limited scalability
  • Bus contention may occur

Example

  • Symmetric Multiprocessor (SMP)

Conclusion

UMA Architecture provides uniform access to memory for all processors, making system design simple.
