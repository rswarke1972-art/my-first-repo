Symmetric Multiprocessor (SMP)

A Symmetric Multiprocessor (SMP) system is a type of closely coupled multiprocessor system in which all processors share the same memory and have equal access rights.

Features

  • Shared main memory
  • Common operating system
  • Equal access to I/O devices
  • Processors are identical

Working

  • All processors run the same OS
  • Tasks are distributed among processors
  • Processors communicate through shared memory

Advantages

  • High performance
  • Simple communication
  • Better resource utilization

Disadvantages

  • Limited scalability
  • Memory access contention

Conclusion

SMP systems improve performance by allowing multiple processors to work together using shared memory.
