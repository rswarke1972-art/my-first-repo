Closely Coupled Multiprocessor Systems

A Closely Coupled Multiprocessor System is a system in which multiple processors share a common main memory and are connected through a high-speed bus.

Features

  • Shared main memory
  • Common system bus
  • High-speed communication between processors

Advantages

  • Fast data sharing
  • Better performance
  • Efficient resource utilization

Disadvantages

  • Limited scalability
  • Bus contention may occur

Example

  • Symmetric Multiprocessor (SMP) systems

Conclusion

Closely Coupled Systems allow processors to communicate quickly through shared memory, improving overall performance.
