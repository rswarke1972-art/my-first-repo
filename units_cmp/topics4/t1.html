Segmentation

Segmentation is a memory management technique used in 8086 microprocessor to divide memory into different segments. It allows efficient use of memory and supports modular programming.

Need for Segmentation

  • 8086 has 20 bit address bus (1 MB memory)
  • Registers are only 16 bit
  • Segmentation helps access 1 MB memory using 16 bit registers

Types of Segments

  • Code Segment (CS) – Stores program instructions
  • Data Segment (DS) – Stores data
  • Stack Segment (SS) – Stores stack data
  • Extra Segment (ES) – Additional data segment

Segment Size

  • Each segment can be maximum 64 KB
  • Segment registers hold starting address of segment

Advantages

  • Efficient memory utilization
  • Supports modular programming
  • Improves program organization

Conclusion

Segmentation allows 8086 to access large memory by dividing it into logical sections. It plays an important role in memory organization.
