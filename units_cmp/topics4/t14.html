Software Interrupts

Software Interrupts are interrupts generated by executing a special instruction in a program. They are used to request specific system services.

INT Instruction

  • INT n – Generates interrupt of type n
  • Example: INT 21H

Working

  • Program executes INT instruction
  • Processor saves flags, CS, and IP
  • Control transfers to corresponding ISR
  • Returns using IRET instruction

Uses

  • System calls
  • DOS services
  • Program termination

Advantages

  • Easy to use
  • Provides controlled access to system functions

Conclusion

Software Interrupts allow programs to request system services by using the INT instruction.
