Hardware Interrupts

Hardware Interrupts are signals generated by external devices to request the processor's attention. These interrupts occur due to hardware events.

Types of Hardware Interrupts in 8086

1. INTR (Interrupt Request)

  • Maskable interrupt
  • Can be enabled or disabled using IF flag

2. NMI (Non-Maskable Interrupt)

  • Non-maskable interrupt
  • Cannot be disabled
  • Used for critical errors

Working

  • External device sends interrupt signal
  • Processor completes current instruction
  • Processor saves flags, CS, and IP
  • Control transfers to ISR

Advantages

  • Allows quick response to external events
  • Improves system efficiency

Conclusion

Hardware Interrupts enable external devices to communicate with the processor and request immediate attention.
