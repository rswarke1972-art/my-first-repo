Block Diagram of 8259

The block diagram of 8259 shows the internal functional units used to manage and control interrupt requests.

Main Blocks of 8259

1. Interrupt Request Register (IRR)

  • Stores incoming interrupt requests

2. Interrupt Mask Register (IMR)

  • Used to mask (disable) specific interrupts

3. In-Service Register (ISR)

  • Stores interrupts currently being serviced

4. Priority Resolver

  • Determines priority among multiple interrupts

5. Data Bus Buffer

  • Connects 8259 to system data bus

6. Control Logic

  • Manages overall operation of 8259

Working

  • Interrupt request enters IRR
  • Priority resolver selects highest priority interrupt
  • Processor is notified

Conclusion

The block diagram of 8259 explains how different internal components work together to manage interrupt handling.
