I/O Mapped I/O

In I/O Mapped I/O, input and output devices are assigned a separate address space different from memory. Special instructions are used to access these devices.

Working

  • I/O devices are given port addresses
  • Uses IN and OUT instructions
  • M/IO control signal distinguishes I/O operation

Features

  • Uses 16 bit port address space
  • Separate from memory address space
  • Maximum 64K I/O ports

Advantages

  • Does not reduce memory space
  • Clear separation between memory and I/O

Disadvantages

  • Limited instruction set (IN and OUT only)

Conclusion

I/O Mapped I/O provides a separate address space for I/O devices, ensuring memory space remains fully available.
