Logical to Physical Address Translation

In 8086 microprocessor, the logical address is converted into a physical address to access memory. The logical address consists of a segment address and an offset address.

Logical Address

  • Segment Address : Offset
  • Example: 2000H : 1000H

Physical Address Calculation

Physical Address = (Segment Address × 10H) + Offset

Example Calculation

  • Segment = 2000H
  • Offset = 1000H
  • Physical Address = 20000H + 1000H = 21000H

Explanation

  • Segment address is shifted left by 4 bits (multiplied by 10H)
  • Offset is added to get final 20 bit physical address
  • Allows access up to 1 MB memory

Conclusion

Logical to Physical Address Translation enables 8086 to access 1 MB memory using 16 bit segment and offset registers.
