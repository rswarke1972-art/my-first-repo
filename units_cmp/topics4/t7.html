Memory Mapped I/O

In Memory Mapped I/O, input and output devices are assigned addresses within the memory address space. The processor treats I/O devices like memory locations.

Working

  • I/O devices are given memory addresses
  • Normal memory instructions (MOV, ADD, etc.) are used
  • M/IO control signal is used to distinguish memory operation

Features

  • Uses 20 bit address bus
  • Can access up to 1 MB address space
  • All memory instructions can be used

Advantages

  • Flexible programming
  • No separate I/O instructions required

Disadvantages

  • Reduces available memory space

Conclusion

Memory Mapped I/O allows I/O devices to be accessed like memory, making programming easier but reducing available memory space.
