Interrupt Service Routine (ISR)

An Interrupt Service Routine (ISR) is a special program that is executed when an interrupt occurs. It handles the interrupt and performs the required task.

Working of ISR

  • Interrupt occurs
  • Processor saves current execution state
  • Control transfers to ISR using IVT
  • ISR executes required task
  • Processor returns to main program using IRET

Important Instruction

  • IRET – Interrupt Return instruction

Features

  • Handles hardware or software interrupts
  • Must end with IRET instruction
  • Stored in memory with segment and offset address

Conclusion

ISR ensures proper handling of interrupts by temporarily pausing the main program and executing a specific routine.
