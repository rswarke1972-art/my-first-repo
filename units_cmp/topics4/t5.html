Address Mapping

Address Mapping is the process of assigning memory or I/O devices to specific address ranges so that the microprocessor can identify and access them correctly.

Purpose of Address Mapping

  • To connect memory and I/O devices to processor
  • To assign unique address ranges
  • To avoid address conflicts

Types of Address Mapping

1. Memory Address Mapping

  • Memory devices are assigned specific address ranges
  • Processor uses address bus to select memory location

2. I/O Address Mapping

  • I/O devices are assigned port addresses
  • Uses IN and OUT instructions

Example

  • ROM: 00000H – 1FFFFH
  • RAM: 20000H – 3FFFFH

Conclusion

Address Mapping ensures proper communication between processor and memory or I/O devices by assigning unique address spaces.
