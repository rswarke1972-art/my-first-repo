Interrupt Vector Table (IVT)

The Interrupt Vector Table (IVT) is a table in memory that stores the starting addresses of all Interrupt Service Routines (ISR). It helps the processor locate the correct ISR when an interrupt occurs.

Location of IVT

  • Located at memory address 00000H
  • Occupies first 1 KB of memory
  • Contains 256 interrupt vectors

Structure

  • Each interrupt vector is 4 bytes
  • First 2 bytes store offset address
  • Next 2 bytes store segment address

Working

  • When interrupt occurs, interrupt type number is generated
  • Processor multiplies type number by 4
  • Fetches ISR address from IVT

Conclusion

The Interrupt Vector Table enables the 8086 to quickly locate and execute the correct Interrupt Service Routine.
