Interrupt Status Register

The Interrupt Status Register stores information about the current status of interrupts in the system. It indicates which interrupts are active or pending.

Purpose

  • Shows pending interrupt requests
  • Indicates currently serviced interrupt
  • Helps in interrupt management

Working

  • Updated when an interrupt request occurs
  • Checked by processor or interrupt controller
  • Works along with interrupt control register

Importance

  • Prevents loss of interrupt signals
  • Ensures proper interrupt servicing
  • Helps maintain system reliability

Conclusion

The Interrupt Status Register helps monitor and manage interrupt activity in the microprocessor system.
