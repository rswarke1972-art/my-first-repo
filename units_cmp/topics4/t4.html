Read Write Cycle Timing Diagram

The Read and Write cycle timing diagram of 8086 shows the sequence of operations during memory or I/O read and write operations. Each operation takes place in clock cycles called T-states.

T-States

  • T1 – Address is placed on address bus
  • T2 – Control signals are activated
  • T3 – Data transfer takes place
  • T4 – Operation completes

Read Cycle

  • Address is sent to memory
  • RD signal becomes active (low)
  • Data is read from memory to processor

Write Cycle

  • Address is sent to memory
  • WR signal becomes active (low)
  • Data is written from processor to memory

Important Signals

  • RD – Read control signal
  • WR – Write control signal
  • M/IO – Memory or I/O selection
  • ALE – Address Latch Enable

Conclusion

The timing diagram explains how the 8086 performs read and write operations step by step using clock cycles and control signals.
