Even and Odd Memory Banks

In 8086, memory is divided into two banks: Even memory bank and Odd memory bank. This arrangement allows the processor to access 16 bit data efficiently.

Even Memory Bank

  • Stores data at even addresses (0, 2, 4, ...)
  • Connected to lower 8 data lines (D0 – D7)

Odd Memory Bank

  • Stores data at odd addresses (1, 3, 5, ...)
  • Connected to higher 8 data lines (D8 – D15)

Working

  • For 8 bit data, only one bank is used
  • For 16 bit data at even address, both banks are used
  • Uses A0 and BHE signals for bank selection

Advantages

  • Improves data transfer speed
  • Efficient 16 bit data access

Conclusion

Even and Odd memory banks allow 8086 to access 16 bit data in a single operation, improving performance.
