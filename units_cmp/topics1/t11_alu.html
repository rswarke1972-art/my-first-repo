Arithmetic and Logic Unit (ALU)

The Arithmetic and Logic Unit (ALU) is the part of the CPU that performs arithmetic and logical operations.

Main Functions of ALU

  • Arithmetic Operations: Addition, subtraction, multiplication, division
  • Logical Operations: AND, OR, NOT, XOR
  • Comparison: Compare two values (>, <, =)

Components of ALU

  • Arithmetic section – Performs calculations
  • Logic section – Performs logical operations
  • Accumulator – Stores temporary results
  • Status register – Stores flags (carry, zero, overflow)

ALU Working

The control unit sends the operation code (opcode) to the ALU. Based on the opcode, ALU performs the operation and sends the result back to the register or memory.

Importance of ALU

ALU is essential for CPU operations as it performs all mathematical and logical tasks required by programs.
