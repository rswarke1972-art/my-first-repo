Bus Interconnection

Bus interconnection is a method of connecting different components of a computer system using a common communication path called a bus. A bus carries data, addresses, and control signals between components.

Main Parts of a Bus

  • Data Bus – Transfers data between CPU and memory.
  • Address Bus – Carries memory addresses.
  • Control Bus – Carries control signals like read/write.

Types of Bus Interconnection

1. Single Bus System

All devices share one bus for data transfer. It is simple but slower due to congestion.

2. Multiple Bus System

Uses multiple buses to improve speed and reduce traffic.

3. Local Bus

A dedicated bus for high-speed devices like graphics cards.

Bus Performance Factors

  • Bus width (number of lines)
  • Bus speed (frequency)
  • Number of devices connected

Conclusion

Bus interconnection is a simple and cost-effective way to connect computer components, but performance depends on bus width and speed.
