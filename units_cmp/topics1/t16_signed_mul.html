Signed Operand Multiplication

Signed multiplication is used when one or both operands are negative. Computers use two’s complement representation for signed numbers.

Steps for Signed Multiplication

  • Convert both operands to two’s complement
  • Multiply as positive numbers
  • If result is negative, convert back using two’s complement

Example

Multiply +5 and -3

  • +5 = 0101
  • -3 = 1101 (two’s complement)
  • Result = 0101 × 1101 = 1011 (−15 in two’s complement)

Important Points

  • Sign bit is the MSB (most significant bit)
  • Negative result is represented in two’s complement
  • Overflow must be checked
