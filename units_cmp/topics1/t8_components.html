Computer Components

A computer system is made up of several components that work together to perform tasks. These components are divided into hardware and software.

Main Computer Components

1. Central Processing Unit (CPU)

CPU is the brain of the computer. It controls all operations and performs calculations and logic operations.

2. Memory

Memory stores data and instructions. It includes: RAM (temporary) and ROM (permanent).

3. Input Devices

Devices used to enter data into a computer. Example: Keyboard, Mouse, Scanner.

4. Output Devices

Devices used to display results. Example: Monitor, Printer, Speaker.

5. Storage Devices

Devices used to store data permanently. Example: HDD, SSD, Pen drive.

6. Motherboard

Main circuit board that connects all components.

Conclusion

All components work together to execute instructions and provide desired results.
