Booth’s Algorithm

Booth’s Algorithm is a fast multiplication technique used for multiplying signed binary numbers. It reduces the number of required addition and subtraction steps.

How Booth’s Algorithm Works

  • It scans the multiplier from right to left.
  • It uses a special register called Q-1 which stores the previous bit.
  • Based on the current bit and Q-1, it decides whether to add, subtract, or do nothing.

Rules

  • If the last two bits are 01 → Add the multiplicand.
  • If the last two bits are 10 → Subtract the multiplicand.
  • If the last two bits are 00 or 11 → Do nothing.

Example

Suppose multiplier = 3 (0011) and multiplicand = -4 (1100). Booth’s algorithm will efficiently perform multiplication using shifts and add/subtract operations.

Advantages

  • Works for signed numbers
  • Reduces number of arithmetic operations
  • Improves multiplication speed

Conclusion

Booth’s Algorithm is widely used in computer arithmetic units for efficient signed multiplication.
