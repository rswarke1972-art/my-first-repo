Addition and Subtraction of Signed Numbers

Signed numbers are numbers with a sign (+ or −). In computers, signed numbers are usually represented using Two’s Complement form.

Two’s Complement Representation

Two’s complement is a method used to represent negative numbers. It is easy to perform arithmetic operations using this method.

Addition of Signed Numbers

While adding two signed numbers:

  • Convert both numbers to two’s complement form
  • Add them using binary addition
  • If carry is generated, ignore the carry bit

Example

Add +5 and −3 using 4-bit representation:

  • +5 = 0101
  • −3 = 1101 (two’s complement)
  • Sum = 0101 + 1101 = 0010 = +2

Subtraction of Signed Numbers

Subtraction is performed by adding the negative value. Example: A − B = A + (−B)

Overflow in Signed Addition

Overflow occurs when the result is outside the range of representation.
