Multiplication of Positive Numbers

Multiplication in computers is done using binary numbers. The process is similar to decimal multiplication but uses binary digits.

Binary Multiplication Method

Each bit of the multiplier is multiplied with the multiplicand. Partial products are generated and shifted based on bit position.

Example (4-bit)

Multiply 0101 (5) × 0011 (3)

  • Step 1: 0101 × 1 = 0101
  • Step 2: 0101 × 1 (shift left 1) = 1010
  • Step 3: Add partial products = 0101 + 1010 = 1111 (15)

Shift and Add Method

Multiplication can be implemented using shift and add logic in ALU. It is fast and easy to design.

Conclusion

Multiplication of positive numbers is a basic function of ALU. It is performed using binary multiplication and shift-add method.
