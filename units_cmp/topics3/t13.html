Data Movement Instructions

Data Movement Instructions in 8086 are used to transfer data from one location to another. The data can be moved between registers, memory, and I/O devices.

Common Data Movement Instructions

1. MOV (Move)

  • Transfers data from source to destination
  • Example: MOV AX, BX

2. PUSH

  • Stores data onto the stack
  • Example: PUSH AX

3. POP

  • Removes data from the stack
  • Example: POP AX

4. XCHG

  • Exchanges data between two operands
  • Example: XCHG AX, BX

5. IN and OUT

  • Used for input and output operations
  • Example: IN AL, 60H

Features

  • Does not affect flags (mostly)
  • Used frequently in assembly programs

Conclusion

Data Movement Instructions are essential for transferring data between registers, memory, stack, and I/O ports in 8086.
