Indirect Addressing Mode

In Indirect Addressing Mode, the address of the operand is stored in a register. The instruction refers to the register, and the actual memory address is taken from that register.

Format

Instruction Register, [Register]

Example

  • MOV AX, [BX]
  • MOV AL, [SI]

In the above examples, BX and SI contain the memory address of the data.

Features

  • Memory address is stored in a register
  • Flexible addressing method
  • Requires memory access

Advantages

  • Allows dynamic memory access
  • Useful for arrays and pointers

Conclusion

Indirect Addressing Mode provides flexibility by storing the memory address inside a register instead of directly in the instruction.
