Based Addressing Mode

In Based Addressing Mode, the effective address of the operand is obtained by adding a base register to a displacement value. It is commonly used for accessing data structures.

Format

Instruction Register, [Base Register + Displacement]

Example

  • MOV AX, [BX + 2000H]
  • MOV AL, [BP + 10H]

In the above examples, BX or BP acts as the base register. The displacement is added to calculate the effective address.

Features

  • Uses base registers (BX or BP)
  • Effective address calculated during execution
  • Useful for structured data access

Advantages

  • Flexible memory addressing
  • Efficient for handling data structures

Conclusion

Based Addressing Mode provides flexibility by using base registers to calculate memory addresses dynamically.
