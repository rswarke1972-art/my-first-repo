Control Transfer Instructions

Control Transfer Instructions in 8086 are used to change the normal sequence of execution of a program. These instructions transfer control from one part of the program to another.

Types of Control Transfer Instructions

1. Unconditional Jump

  • JMP – Transfers control to specified address
  • Example: JMP 2000H

2. Conditional Jump

  • JZ – Jump if Zero
  • JNZ – Jump if Not Zero
  • JC – Jump if Carry
  • Example: JZ LABEL

3. CALL Instruction

  • Calls a procedure
  • Saves return address in stack

4. RET Instruction

  • Returns control from procedure

5. LOOP Instruction

  • Repeats a block of code
  • Uses CX register as counter

Importance

  • Used in decision making
  • Used in loops and procedures
  • Controls program flow

Conclusion

Control Transfer Instructions help in branching, looping, and procedure calls, making program execution flexible and structured.
