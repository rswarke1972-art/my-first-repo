Pin Diagram of 8086

The 8086 microprocessor has 40 pins. These pins are used for data transfer, addressing, control signals, and power supply connections.

Classification of Pins

1. Address Bus

  • 20 address lines (A0 – A19)
  • Used to access 1 MB memory

2. Data Bus

  • 16 data lines (D0 – D15)
  • Used for data transfer
  • Multiplexed with lower address lines (AD0 – AD15)

3. Control Signals

  • RD (Read)
  • WR (Write)
  • M/IO (Memory or I/O selection)
  • INTA (Interrupt Acknowledge)

4. Power Supply

  • Vcc – +5V power supply
  • GND – Ground

5. Clock and Other Signals

  • CLK – Clock input
  • RESET – Resets the processor
  • READY – Indicates readiness of device

Minimum and Maximum Mode

8086 operates in two modes:

  • Minimum Mode – Single processor system
  • Maximum Mode – Multi-processor system

Conclusion

The pin diagram of 8086 shows how the processor connects with memory and external devices. Each pin has a specific function that helps in communication and control.
