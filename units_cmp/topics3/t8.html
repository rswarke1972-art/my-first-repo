Direct Addressing Mode

In Direct Addressing Mode, the instruction contains the memory address of the operand. The data is stored in memory, and the address is directly specified in the instruction.

Format

Instruction Register, [Memory Address]

Example

  • MOV AX, [2000H]
  • ADD AL, [3000H]

In the above examples, 2000H and 3000H are direct memory addresses.

Features

  • Memory address is directly given
  • Requires memory access
  • Slower than register addressing mode

Advantages

  • Can access specific memory locations
  • Simple to understand

Conclusion

Direct Addressing Mode allows access to a specific memory location by directly mentioning its address in the instruction.
