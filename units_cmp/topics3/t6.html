Immediate Addressing Mode

In Immediate Addressing Mode, the operand (data) is given directly in the instruction itself. The data is a constant value and does not refer to any memory location.

Format

Instruction Destination, Data

Example

  • MOV AX, 1234H
  • MOV BL, 25H

In the above examples, 1234H and 25H are immediate data values.

Features

  • Operand is part of instruction
  • No memory address is required
  • Fast execution

Advantages

  • Simple and easy to use
  • Faster than memory addressing modes

Conclusion

Immediate Addressing Mode is used when constant data is required in the instruction. It provides faster execution because no memory access is needed.
