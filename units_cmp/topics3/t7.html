Register Addressing Mode

In Register Addressing Mode, the operand is stored in a register. The instruction specifies the register that contains the data.

Format

Instruction Destination, Register

Example

  • MOV AX, BX
  • ADD AX, CX

In the above examples, BX and CX are registers that contain the data.

Features

  • Operand is stored in register
  • No memory access required
  • Faster execution compared to memory addressing modes

Advantages

  • Very fast operation
  • Simple instruction format

Conclusion

Register Addressing Mode improves performance by using CPU registers to store operands, reducing the need for memory access.
