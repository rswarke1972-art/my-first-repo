String Instructions

String Instructions in 8086 are used to perform operations on strings or arrays of data. These instructions automatically use SI (Source Index) and DI (Destination Index) registers.

Common String Instructions

1. MOVS

  • Moves data from source to destination
  • Uses SI and DI registers

2. LODS

  • Loads string data into accumulator

3. STOS

  • Stores accumulator data into memory

4. CMPS

  • Compares two strings

5. SCAS

  • Scans string for a value

Important Registers Used

  • SI – Source Index
  • DI – Destination Index
  • CX – Counter register

Features

  • Useful for array operations
  • Can be repeated using REP prefix

Conclusion

String Instructions simplify operations on arrays and strings by automatically handling index registers in 8086.
