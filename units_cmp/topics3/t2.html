Architecture of 8086

The 8086 microprocessor is a 16 bit processor developed by Intel. Its architecture is divided into two main units: Bus Interface Unit (BIU) and Execution Unit (EU).

Main Components of 8086

1. Bus Interface Unit (BIU)

  • Handles communication with memory and I/O devices
  • Fetches instructions from memory
  • Contains segment registers and instruction queue

2. Execution Unit (EU)

  • Executes instructions
  • Performs arithmetic and logic operations
  • Contains ALU, general purpose registers and flag register

Important Features

  • 16 bit data bus
  • 20 bit address bus
  • Can access up to 1 MB memory
  • Uses pipelining (instruction queue of 6 bytes)

Block Diagram Explanation

The BIU and EU work in parallel. While the Execution Unit executes an instruction, the Bus Interface Unit fetches the next instruction. This improves processing speed.

Conclusion

The 8086 architecture improves performance by dividing tasks between BIU and EU. This design increases efficiency and speed of execution.
