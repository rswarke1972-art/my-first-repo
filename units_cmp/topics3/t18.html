Input / Output Instructions

Input / Output (I/O) Instructions in 8086 are used to transfer data between the microprocessor and external devices such as keyboard, printer, and other peripherals.

Main I/O Instructions

1. IN Instruction

  • Transfers data from input device to accumulator
  • Example: IN AL, 60H

2. OUT Instruction

  • Transfers data from accumulator to output device
  • Example: OUT 40H, AL

Working

  • Uses specific I/O port addresses
  • Data is transferred through data bus
  • Usually involves AL or AX register

Features

  • Used for communication with external hardware
  • Simple instruction format

Conclusion

Input and Output Instructions allow 8086 to interact with external devices by sending and receiving data through I/O ports.
