Registers of 8086

The 8086 microprocessor contains 14 registers. These registers are used to store data, addresses, and intermediate results during execution. All general purpose registers are 16 bit in size.

Types of Registers

1. General Purpose Registers

  • AX (Accumulator) – Used in arithmetic and logic operations
  • BX (Base Register) – Used for base addressing
  • CX (Count Register) – Used in loop and shift operations
  • DX (Data Register) – Used in multiplication and division

Each of these registers can be divided into two 8 bit parts:

  • AH and AL
  • BH and BL
  • CH and CL
  • DH and DL

2. Segment Registers

  • CS – Code Segment
  • DS – Data Segment
  • SS – Stack Segment
  • ES – Extra Segment

3. Pointer and Index Registers

  • SP – Stack Pointer
  • BP – Base Pointer
  • SI – Source Index
  • DI – Destination Index

4. Special Registers

  • IP – Instruction Pointer
  • Flag Register – Contains status and control flags

Conclusion

Registers of 8086 are essential for storing data and controlling program execution. They improve processing speed by reducing memory access.
