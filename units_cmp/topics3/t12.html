Based Indexed Addressing Mode

In Based Indexed Addressing Mode, the effective address of the operand is calculated by adding a base register and an index register. It may also include a displacement.

Format

Instruction Register, [Base Register + Index Register + Displacement]

Example

  • MOV AX, [BX + SI]
  • MOV AL, [BP + DI + 10H]

In the above examples, BX or BP acts as the base register, SI or DI acts as the index register, and displacement is optional.

Features

  • Uses both base and index registers
  • Can include displacement
  • Very flexible addressing mode

Advantages

  • Efficient for complex data structures
  • Useful for multi-dimensional arrays

Conclusion

Based Indexed Addressing Mode provides maximum flexibility in address calculation by combining base register, index register, and displacement.
