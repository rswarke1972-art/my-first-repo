Indexed Addressing Mode

In Indexed Addressing Mode, the effective address of the operand is calculated by adding an index register to a displacement value. It is mainly used for accessing elements of arrays.

Format

Instruction Register, [Index Register + Displacement]

Example

  • MOV AX, [SI + 1000H]
  • MOV AL, [DI + 05H]

In the above examples, SI or DI acts as the index register, and the displacement is added to get the effective address.

Features

  • Uses index registers (SI or DI)
  • Supports array operations
  • Effective address is calculated during execution

Advantages

  • Efficient for accessing array elements
  • Flexible memory access

Conclusion

Indexed Addressing Mode is useful for handling arrays and data structures by calculating memory addresses dynamically.
