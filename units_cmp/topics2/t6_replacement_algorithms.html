Replacement Algorithms

When the cache is full and a new block needs to be loaded, the system must decide which existing block should be removed. The rule used for this decision is called a replacement algorithm.

Common Replacement Algorithms

1. Least Recently Used (LRU)

LRU removes the block that has not been used for the longest time. It assumes that recently used blocks will be used again soon.

2. First In First Out (FIFO)

FIFO removes the oldest block in cache (first stored). It is simple but may remove frequently used data.

3. Random Replacement

Randomly selects a block to remove. Simple but not always efficient.

4. Least Frequently Used (LFU)

Removes the block that is used least frequently. It counts how many times each block is used.

Conclusion

Replacement algorithms help improve cache performance. LRU is the most commonly used algorithm in modern systems.
