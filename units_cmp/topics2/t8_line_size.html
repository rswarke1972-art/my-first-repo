Line Size

Line size (also called block size) is the amount of data stored in one cache line. It is an important factor that affects cache performance.

How Line Size Works

Cache stores data in blocks. When a memory block is loaded into cache, the entire block (line) is copied, not just one byte.

Effect of Line Size

  • Small line size → fewer bytes per block
  • Large line size → more bytes per block

Advantages of Large Line Size

  • Better use of spatial locality
  • Fewer cache misses if nearby data is needed

Disadvantages of Large Line Size

  • Wastes space if nearby data is not used
  • Increases memory transfer time

Conclusion

Line size should be chosen carefully to balance between cache hit rate and memory usage.
