Two Level Cache

A two level cache system uses two separate cache memories to improve performance. These are called Level 1 (L1) cache and Level 2 (L2) cache.

Need for Two Level Cache

As processor speed increased, a single cache was not enough to provide data at the required speed. To reduce the average memory access time, a second cache level was introduced.

Level 1 (L1) Cache

  • Smallest and fastest cache memory
  • Located inside the CPU
  • Very low access time
  • Stores frequently used instructions and data

Level 2 (L2) Cache

  • Larger than L1 cache
  • Slower than L1 but faster than main memory
  • May be inside or outside the CPU
  • Acts as backup cache for L1

Working of Two Level Cache

When the CPU needs data:

  • First, it checks L1 cache.
  • If data is found (cache hit), it is used immediately.
  • If not found (cache miss), it checks L2 cache.
  • If still not found, data is fetched from main memory.

Advantages

  • Improves system performance
  • Reduces average memory access time
  • Better handling of large programs

Conclusion

Two level cache systems improve efficiency by combining a small fast cache (L1) with a larger slower cache (L2), reducing the need to access main memory frequently.
