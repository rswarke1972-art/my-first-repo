Number of Cache Levels

Modern computer systems use multiple levels of cache memory. Each level differs in size, speed, and distance from the CPU.

Cache Levels

  • L1 Cache – Smallest and fastest cache located inside CPU
  • L2 Cache – Larger than L1 but slower, usually inside CPU
  • L3 Cache – Even larger and slower, shared between CPU cores

Why Multiple Cache Levels?

A single cache cannot satisfy both high speed and large capacity. Multiple levels provide a balance between speed and size.

How Cache Levels Work

When CPU searches for data:

  • If found in L1 → Fastest
  • If not found → search L2
  • If not found → search L3
  • If still not found → access main memory

Conclusion

Multi-level cache improves performance by providing fast access to frequently used data.
