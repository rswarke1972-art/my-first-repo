One Level Cache

One level cache means the computer system uses only one cache memory between the CPU and main memory.

Characteristics

  • Only one cache level (usually L1)
  • Fast access time
  • Limited cache size
  • Simple design

How It Works

CPU requests data → cache is checked → if data exists, it is a cache hit. If not, data is fetched from main memory and stored in cache.

Advantages

  • Simple and easy to implement
  • Lower cost
  • Fast access

Disadvantages

  • Limited capacity
  • Higher miss rate compared to multi-level cache
  • Not suitable for modern high-performance systems

Conclusion

One level cache is used in basic systems but modern computers use multiple cache levels for better performance.
