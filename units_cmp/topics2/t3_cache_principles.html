Cache Memory Principles

Cache memory is a small and very fast memory placed between the CPU and main memory. It stores frequently used data and instructions to reduce access time and improve performance.

Main Principles of Cache Memory

1. Temporal Locality

Recently accessed data is likely to be accessed again soon. Cache stores these data so they can be reused quickly.

2. Spatial Locality

Data located near recently accessed data is likely to be accessed soon. Cache stores blocks of data instead of single bytes.

3. Cache Hit

When the CPU finds requested data in cache, it is called a cache hit. Hit reduces memory access time significantly.

4. Cache Miss

When the data is not found in cache, it is called a cache miss. Data must be fetched from main memory, causing delay.

Cache Block

Cache stores data in blocks (also called lines). Each block contains multiple bytes of data.

Conclusion

Cache memory improves computer performance by storing frequently used data and reducing the time needed to access main memory.
