Hard Disk Organization

A hard disk is a secondary storage device used to store large amounts of data permanently. It stores data magnetically on rotating disks called platters.

Main Components of Hard Disk

  • Platters – Circular magnetic disks that store data
  • Spindle – Rotates the platters at high speed
  • Read/Write Head – Reads and writes data on the disk
  • Actuator Arm – Moves the read/write head across the disk

Disk Structure

  • Tracks – Circular paths on the surface of the platter
  • Sectors – Smallest storage units on a track
  • Cylinders – Collection of tracks aligned vertically across platters

Access Time

The total time required to read or write data consists of:

  • Seek Time – Time to move the head to the correct track
  • Rotational Latency – Time for the sector to rotate under the head
  • Transfer Time – Time to transfer data

Advantages

  • Large storage capacity
  • Non-volatile memory
  • Cost effective for bulk storage

Conclusion

Hard disk organization allows efficient storage and retrieval of large amounts of data. It is widely used as secondary storage in computer systems.
