IPv4 (Internet Protocol Version 4)

IPv4 is the fourth version of the Internet Protocol, used to identify devices on a network through an IP address.

Features

  • 32-bit address
  • Supports about 4.3 billion addresses
  • Uses dotted decimal format (e.g., 192.168.1.1)

Limitations

  • Limited address space
  • Security not built-in

Conclusion

IPv4 enables device identification and communication over networks.
