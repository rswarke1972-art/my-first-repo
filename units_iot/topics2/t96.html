Raspberry Pi

Raspberry Pi is a small, low-cost single-board computer used for IoT and embedded applications.

Features

  • Runs Linux operating system
  • Supports HDMI, USB, Ethernet
  • GPIO pins for hardware control

Applications

  • Smart home systems
  • IoT gateway
  • Educational projects

Conclusion

Raspberry Pi provides computing power for advanced IoT projects.
