Representation using Matrices

A relation on a finite set can be represented using a matrix. This method helps analyze properties of relations easily.

Matrix Representation

  • If A = {a₁, a₂, ..., aₙ}
  • Create an n × n matrix M
  • M[i][j] = 1 if (aᵢ, aⱼ) ∈ R
  • M[i][j] = 0 otherwise

Example

A = {1, 2} R = {(1,1), (1,2)}

Matrix: [1 1] [0 0]

Advantages

  • Easy to compute transitive closure
  • Useful in algorithms

Conclusion

Matrix representation provides a structured way to analyze relations.
