Warshall's Algorithm

Warshall's Algorithm is used to find the transitive closure of a relation. It works using matrix representation.

Steps

  • Start with adjacency matrix of relation
  • Update matrix using intermediate vertices
  • Continue until all vertices are considered

Formula

M[i][j] = M[i][j] OR (M[i][k] AND M[k][j])

Applications

  • Finding reachability in graphs
  • Database query processing

Conclusion

Warshall's Algorithm efficiently computes the transitive closure of a relation.
