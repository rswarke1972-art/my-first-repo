Recursive Functions

A recursive function is defined in terms of itself. It solves a problem by breaking it into smaller subproblems.

Main Components

  • Base Case – Stops the recursion
  • Recursive Step – Function calls itself

Example: Factorial Function

n! = n × (n - 1)! Base Case: 0! = 1

Example in Programming

function factorial(n) { if (n == 0) return 1; return n * factorial(n - 1); }

Applications

  • Tree traversal
  • Mathematical sequences
  • Divide and conquer algorithms

Conclusion

Recursive functions solve problems by repeatedly calling themselves until a base condition is met.
