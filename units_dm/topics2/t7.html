Partitions

A partition of a set is a division of the set into non-empty, disjoint subsets such that every element belongs to exactly one subset.

Definition

  • Subsets are non-empty
  • Subsets are disjoint
  • Union of all subsets equals the original set

Example

A = {1, 2, 3, 4}

Possible Partition: {1, 2} , {3, 4}

Relation with Equivalence Relation

  • Every equivalence relation defines a partition
  • Every partition defines an equivalence relation

Conclusion

Partitions divide a set into distinct and non-overlapping groups.
