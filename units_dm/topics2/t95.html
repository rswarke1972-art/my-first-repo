Composition of Functions

Composition of functions means applying one function to the result of another function. It is denoted by (f ∘ g)(x).

Definition

  • If f: A → B and g: B → C
  • Then (g ∘ f)(x) = g(f(x))

Example

Let f(x) = x + 2 Let g(x) = 2x

(g ∘ f)(x) = g(f(x)) = g(x + 2) = 2(x + 2)

Properties

  • Not always commutative
  • Associative

Conclusion

Function composition combines two functions into a single function.
