Inverse of Functions

The inverse of a function reverses the mapping of the original function. It is denoted as f⁻¹.

Condition for Inverse

  • Function must be bijective
  • Each output corresponds to exactly one input

Definition

  • If f(a) = b, then f⁻¹(b) = a

Example

Let f(x) = 2x + 3

To find inverse: y = 2x + 3 x = (y - 3) / 2 So, f⁻¹(x) = (x - 3) / 2

Property

  • f(f⁻¹(x)) = x
  • f⁻¹(f(x)) = x

Conclusion

Inverse functions undo the effect of the original function.
