Types of Functions (Injective, Surjective, Bijective)

Functions can be classified based on how elements of the domain map to elements of the codomain.

1. Injective Function (One-to-One)

  • Each element of domain maps to a unique element
  • No two elements share the same image

2. Surjective Function (Onto)

  • Every element of codomain has a pre-image
  • Range equals codomain

3. Bijective Function

  • Both injective and surjective
  • One-to-one correspondence

Importance

  • Used in defining inverse functions
  • Important in counting and mappings

Conclusion

Functions are categorized based on their mapping behavior between sets.
