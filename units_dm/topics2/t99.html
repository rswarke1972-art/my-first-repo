Counting Functions

Counting functions are used to determine the number of possible functions between two sets.

Number of Functions

  • If |A| = m and |B| = n
  • Total number of functions from A to B = n^m

Injective Functions

  • Possible only if m ≤ n
  • Number = nPm (permutations)

Surjective Functions

  • Every element of B must have pre-image

Example

If A has 2 elements and B has 3 elements, number of functions = 3² = 9

Conclusion

Counting functions helps analyze possible mappings between sets.
