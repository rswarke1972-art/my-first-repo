Representation using Digraphs

A relation on a set can also be represented using a directed graph (digraph).

Digraph Representation

  • Each element of the set is represented as a node
  • An arrow (directed edge) from a to b represents (a, b) ∈ R

Example

If A = {1, 2, 3} R = {(1,2), (2,3)}

  • Draw arrow from 1 → 2
  • Draw arrow from 2 → 3

Advantages

  • Visual representation
  • Easy to understand properties like transitivity

Conclusion

Digraphs provide a graphical way to represent and analyze relations.
