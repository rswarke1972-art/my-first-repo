Properties of Relations

A relation on a set may satisfy certain properties that describe its behavior.

Main Properties

1. Reflexive

  • For every element a in set A, (a, a) ∈ R

2. Symmetric

  • If (a, b) ∈ R, then (b, a) ∈ R

3. Antisymmetric

  • If (a, b) ∈ R and (b, a) ∈ R, then a = b

4. Transitive

  • If (a, b) ∈ R and (b, c) ∈ R, then (a, c) ∈ R

Importance

  • Helps classify relations
  • Used in equivalence and ordering

Conclusion

Understanding properties of relations is essential for studying equivalence and order relations.
