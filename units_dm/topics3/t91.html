Prim's Algorithm

Prim's Algorithm is used to find the Minimum Spanning Tree (MST) of a weighted connected graph.

Main Idea

  • Start from any vertex
  • Select the minimum weight edge that connects a new vertex
  • Repeat until all vertices are included

Steps

  • Initialize with starting vertex
  • Choose smallest edge from visited to unvisited
  • Add selected edge to MST

Application

  • Network design

Conclusion

Prim's Algorithm builds MST by expanding one vertex at a time.
