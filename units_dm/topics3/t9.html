Minimum Spanning Tree

A Minimum Spanning Tree (MST) is a spanning tree of a connected weighted graph that has the minimum total edge weight.

Spanning Tree

  • Includes all vertices of the graph
  • Contains no cycles
  • Has n - 1 edges for n vertices

Minimum Spanning Tree

  • Spanning tree with smallest total weight

Applications

  • Network design
  • Road construction
  • Telecommunication systems

Conclusion

MST connects all vertices with minimum cost.
