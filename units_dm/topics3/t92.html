Kruskal's Algorithm

Kruskal's Algorithm is used to find the Minimum Spanning Tree (MST) of a weighted connected graph.

Main Idea

  • Sort all edges in increasing order of weight
  • Select smallest edge that does not form a cycle
  • Repeat until n - 1 edges are selected

Steps

  • Arrange edges by weight
  • Pick smallest edge
  • Check for cycle
  • Add edge if no cycle forms

Difference from Prim's

  • Prim's grows from a vertex
  • Kruskal's grows from smallest edges

Conclusion

Kruskal's Algorithm builds MST by selecting globally smallest edges.
