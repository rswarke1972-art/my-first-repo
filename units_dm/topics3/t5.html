Huffman Coding

Huffman Coding is a data compression technique that uses variable-length prefix codes based on frequency of characters.

Main Idea

  • More frequent characters get shorter codes
  • Less frequent characters get longer codes

Steps

  • Find frequency of each character
  • Create leaf nodes
  • Combine lowest frequency nodes
  • Repeat until one tree remains

Advantages

  • Efficient compression
  • No information loss

Application

  • File compression (ZIP, JPEG)

Conclusion

Huffman coding provides optimal prefix codes for data compression.
