Semi Groups

A Semigroup is an algebraic structure consisting of a non-empty set with an associative binary operation.

Definition

  • Set S with binary operation *
  • Closure: For all a, b ∈ S, a * b ∈ S
  • Associativity: (a * b) * c = a * (b * c)

Example

(Set of integers, +)

Note

  • Identity element is not required

Conclusion

A semigroup requires only closure and associativity.
