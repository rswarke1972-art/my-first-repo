Monoids

A Monoid is a semigroup with an identity element.

Definition

  • Set S with binary operation *
  • Closure: a * b ∈ S
  • Associativity: (a * b) * c = a * (b * c)
  • Identity Element: There exists e ∈ S such that a * e = e * a = a

Example

(Set of integers, +)

  • Identity element is 0

Conclusion

A monoid is a semigroup with an identity element.
