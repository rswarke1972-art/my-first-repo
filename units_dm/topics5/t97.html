Congruence Relations

A Congruence Relation is an equivalence relation that is compatible with a given algebraic operation.

Definition

  • It is an equivalence relation
  • It preserves the operation of the structure

Example (Modulo Arithmetic)

a ≡ b (mod n)

  • n divides (a - b)

Properties

  • Reflexive
  • Symmetric
  • Transitive

Conclusion

Congruence relations allow grouping elements based on modular equivalence.
