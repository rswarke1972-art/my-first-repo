Multiplication Principle

The Multiplication Principle states that if one task can be done in m ways and another independent task can be done in n ways, then both tasks together can be done in m × n ways.

Condition

  • Tasks must be performed sequentially

Example

If you have 3 shirts and 2 pants, you can form 3 × 2 = 6 outfits.

Conclusion

Multiplication principle is used when tasks occur in sequence.
