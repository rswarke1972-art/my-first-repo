Integral Domains

An Integral Domain is a commutative ring with identity and no zero divisors.

Definition

  • Commutative under multiplication
  • Has multiplicative identity (1)
  • No zero divisors: If ab = 0, then either a = 0 or b = 0

Example

Set of integers (Z)

Non-Example

Set of integers modulo 6 (Z₆) contains zero divisors.

Conclusion

Integral domains prevent multiplication from producing zero unless one factor is zero.
