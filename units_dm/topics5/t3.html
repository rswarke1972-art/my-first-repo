Permutations

A permutation is an arrangement of objects where order matters.

Formula

  • Number of permutations of n distinct objects: n! (n factorial)

Permutation of r objects from n

nPr = n! / (n - r)!

Example

Number of ways to arrange 3 books from 5 books:

5P3 = 5! / (5 - 3)! = 5! / 2! = 60

Conclusion

Permutations count arrangements where order is important.
