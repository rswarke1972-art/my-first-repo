Propositional Logic

Propositional Logic deals with statements that are either true or false. These statements are called propositions.

Proposition

  • A declarative statement
  • Has a truth value (True or False)
  • Example: "2 + 2 = 4"

Logical Connectives

  • Negation (¬p)
  • Conjunction (p ∧ q)
  • Disjunction (p ∨ q)
  • Implication (p → q)
  • Biconditional (p ↔ q)

Truth Table

  • Shows truth values of compound statements

Importance

  • Used in programming logic
  • Basis of digital circuits

Conclusion

Propositional Logic helps in logical reasoning and forms the basis of computer science.
