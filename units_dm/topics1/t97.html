Power Set

The Power Set of a set A is the set of all possible subsets of A. It is denoted by P(A).

Definition

  • If A is a set, then P(A) contains every subset of A
  • Includes empty set and A itself

Example

If A = {1, 2}

P(A) = { ∅, {1}, {2}, {1,2} }

Cardinality of Power Set

  • If |A| = n, then |P(A)| = 2ⁿ

Importance

  • Used in combinatorics
  • Important in logic and computer science

Conclusion

Power set represents all possible subsets of a given set.
