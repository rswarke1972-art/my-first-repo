Set Operations

Set operations are used to combine and manipulate sets. They are fundamental concepts in set theory.

Main Set Operations

1. Union (A ∪ B)

  • Elements that belong to A or B

2. Intersection (A ∩ B)

  • Elements common to both A and B

3. Difference (A - B)

  • Elements in A but not in B

4. Complement (A')

  • Elements not in set A

Example

A = {1,2,3} B = {2,3,4}

  • A ∪ B = {1,2,3,4}
  • A ∩ B = {2,3}

Conclusion

Set operations help analyze relationships between sets.
