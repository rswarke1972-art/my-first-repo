Finite and Infinite Sets

A set is classified as finite or infinite based on the number of elements it contains.

Finite Set

  • Contains limited number of elements
  • Has definite cardinality
  • Example: A = {1, 2, 3}

Infinite Set

  • Contains unlimited elements
  • No fixed upper limit
  • Example: Natural numbers

Types of Infinite Sets

  • Countably Infinite
  • Uncountably Infinite

Conclusion

Finite sets have limited elements, while infinite sets extend endlessly.
