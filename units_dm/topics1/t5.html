Proof by Mathematical Induction

Mathematical Induction is a proof technique used to prove statements that are true for all natural numbers.

Steps of Mathematical Induction

1. Basis Step

  • Prove statement is true for n = 1 (or initial value)

2. Inductive Hypothesis

  • Assume statement is true for n = k

3. Inductive Step

  • Prove statement is true for n = k + 1

Example

Prove: 1 + 2 + ... + n = n(n + 1)/2

  • Check for n = 1
  • Assume true for n = k
  • Prove for n = k + 1

Conclusion

Mathematical Induction proves statements for infinite sets of natural numbers.
