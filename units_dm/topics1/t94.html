Diagonalization Argument

The Diagonalization Argument, introduced by Georg Cantor, is used to prove that some sets are uncountable.

Main Idea

  • Assume a list of elements exists
  • Create a new element by changing diagonal values
  • The new element is not in the list

Application

  • Proves real numbers are uncountable
  • Shows infinity has different sizes

Example

Used to prove that real numbers between 0 and 1 cannot be listed completely.

Importance

  • Foundation of set theory
  • Important in theoretical computer science

Conclusion

Diagonalization proves that some infinite sets are larger than others.
