Graph Representation

Graphs can be represented in different ways for efficient storage and processing.

1. Adjacency Matrix

  • 2D matrix of size n × n
  • 1 if edge exists, otherwise 0
  • Suitable for dense graphs

2. Adjacency List

  • Each vertex stores list of adjacent vertices
  • Memory efficient for sparse graphs

3. Edge List

  • List of all edges
  • Stores pairs of connected vertices

Conclusion

Graph representation methods help store and manipulate graph data efficiently.
