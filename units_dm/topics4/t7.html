Hamilton Paths

A Hamilton Path is a path in a graph that visits every vertex exactly once.

Hamilton Circuit

  • A Hamilton Path that starts and ends at the same vertex

Difference from Euler Path

  • Euler Path: Visits every edge once
  • Hamilton Path: Visits every vertex once

Important Note

  • No simple degree-based condition guarantees a Hamilton Path

Conclusion

Hamilton paths focus on visiting each vertex exactly once.
