Dijkstra's Algorithm

Dijkstra's Algorithm is used to find the shortest path from a source vertex to all other vertices in a weighted graph.

Conditions

  • Graph must have non-negative edge weights

Main Idea

  • Start from source vertex
  • Select vertex with smallest distance
  • Update distances of adjacent vertices
  • Repeat until all vertices are visited

Applications

  • GPS navigation systems
  • Network routing

Conclusion

Dijkstra's Algorithm efficiently computes shortest paths in weighted graphs.
