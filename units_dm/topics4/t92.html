Graph Colouring

Graph Colouring is the assignment of colors to vertices of a graph such that no two adjacent vertices have the same color.

Chromatic Number

  • Minimum number of colors needed
  • Denoted by χ(G)

Applications

  • Map coloring
  • Scheduling problems
  • Register allocation in compilers

Example

If a graph is bipartite, it can be colored using only 2 colors.

Conclusion

Graph colouring helps solve conflict and scheduling problems efficiently.
