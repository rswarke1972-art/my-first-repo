Euler Paths

An Euler Path is a path in a graph that visits every edge exactly once.

Euler Circuit

  • An Euler Path that starts and ends at the same vertex

Conditions for Undirected Graph

  • Euler Circuit: All vertices have even degree
  • Euler Path: Exactly two vertices have odd degree

Example

If a graph has 0 or 2 vertices with odd degree, it has an Euler Path.

Conclusion

Euler paths traverse every edge once without repetition.
